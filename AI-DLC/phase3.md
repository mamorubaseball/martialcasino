# AI-DLC - フェーズ 3: データ連携とAI対話機能の統合

本フェーズでは、投資メディアプラットフォームとしての付加価値を最大化するため、ユーザーが記事や銘柄のデータと直接会話できるAIアシスタント機能、および動的・インタラクティブなチャートデータを統合します。

---

## 📋 タスクリスト（計画）

- [ ] **対話型AIアシスタント（Gemini API）の統合**
  - [ ] Firebase AI Logic / Gemini SDK を活用した、記事内チャットコンポーネントの作成
  - [ ] 閲覧中の銘柄財務データや保有ポートフォリオ状況をコンテキストとしてAIに動的に引き渡す機能の実装
  - [ ] セキュアなクライアント側/サーバー側 APIキー管理の確立
- [ ] **インタラクティブなチャート機能の追加**
  - [ ] 静的なグラフ（SVG等）から、Recharts または Chart.js 等を用いた動的チャート（マウスホバーでのデータ詳細表示など）への刷新
  - [ ] 各銘柄の過去データ推移（株価、売上、純利益など）のグラフ表示強化
- [ ] **ユーザーパーソナライズ機能（オプション）**
  - [ ] Firebase Auth を使った簡単な認証と、お気に入り銘柄（ウォッチリスト）の保存機能
  - [ ] 保有アセットの損益分岐アラート機能の統合
