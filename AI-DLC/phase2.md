# AI-DLC - フェーズ 2: SEO・インフラ・実運用最適化

本フェーズでは、作成されたWebサイトを外部公開・実運用するために必要なSEO設定、表示パフォーマンスのチューニング、デプロイ環境の構築、および運用自動化との接続を行います。

---

## 📋 タスクリスト（計画）

- [ ] **SEO & OGP 対策**
  - [ ] 各ブログ記事および銘柄詳細ページでの、メタタグ（Title, Description）とソーシャルシェア用OGPカードの動的最適化
  - [ ] `sitemap.xml` と `robots.txt` の自動生成とクローラ最適化
  - [ ] 構造化データ（JSON-LD）の出力による検索エンジン向け情報構造化
- [ ] **表示・ビルドパフォーマンスの最適化**
  - [ ] 不要なCSS/JSの分離、画像のWebP/AVIF最適化
  - [ ] Lighthouse または Web Vitals（LCP, INP, CLS）スコアの計測と改善（Chrome DevTools 活用）
- [ ] **デプロイ・ホスティング環境の設定**
  - [ ] GitHubリポジトリとホスティングプラットフォーム（Vercel または Cloudflare Pages）の接続設定
  - [ ] ビルドコマンドおよび環境変数の設定
- [ ] **外部データ連携・自動化パイプラインの接続**
  - [ ] マネーフォワード等のポートフォリオ自動更新ワークフロー（`/mf-update` など）から出力されるデータをAstroサイト内に自動反映させる統合実装
