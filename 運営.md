# TechMoney / かぶまる 運営・開発マニュアル

このファイルには、本メディアプラットフォーム（モノレポ）の開発環境起動方法、およびYouTubeから記事を自動インポートする手順をまとめています。

---

## 1. 開発環境の起動・ビルド方法

本プロジェクトは `investment-platform`（ディレクトリ名: `martialcasino`）という1つのリポジトリの下で、`techmoney-web` と `kabumaru-web` の2つのAstroウェブサイトをモノレポ（npm workspaces）として管理しています。

### 事前準備
- **推奨Node.jsバージョン**: `v22.12.0` 以上

### コマンド一覧
ルートディレクトリで以下のコマンドを実行します。

| コマンド | 説明 |
| :--- | :--- |
| `npm run dev:techmoney` | **TechMoney** のローカル開発サーバーを起動（`http://localhost:4321`） |
| `npm run dev:kabumaru` | **かぶまる** のローカル開発サーバーを起動（`http://localhost:4322`） |
| `npm run build:techmoney` | **TechMoney** の本番ビルド（静的ファイル生成） |
| `npm run build:kabumaru` | **かぶまる** の本番ビルド（静的ファイル生成） |

---

## 2. YouTube動画から記事への自動インポート手順

Pythonスクリプト `scripts/import_youtube.py` を用いることで、Google Drive上に保存された動画用アセット（台本・要約HTML）から、Astro用の記事ページとプレースホルダー（`.md`ファイル）を自動生成できます。

### ① Google Driveへのファイル配置ルール
インポート対象のフォルダは、Google Drive内の以下のディレクトリに配置されます。

- **TechMoney用**: `GoogleDrive/その他のパソコン/マイ Mac/techmoney/` 配下の年月フォルダ（例: `202606/`）
- **かぶまる用**: `GoogleDrive/その他のパソコン/マイ Mac/kabu_maru/` 配下の年月フォルダ

#### フォルダ名の命名規則
`{ベース名}+{公開日(YYYYMMDD)}` という形式でフォルダを作成してください。
- 例: `hpe+20260620` （ベース名: `hpe`, 公開日: `2026-06-20`）

#### フォルダ内の構成（必要なファイル）
1. **`video_summary.html`**: AI等で自動生成したレポート形式のHTMLファイル。
2. **`gaiyou.md`** または **`generated_gaiyou.md`**: 動画の概要欄テキスト。
3. **`hpe.md`**（ベース名と同じ名前のファイル）または `chap1.md` 〜 `chapN.md`: トランスクリプト（台本）Markdown。
4. **`url.md` (重要)**: 
   - フォルダ内に `url.md` というテキストファイルを作成し、**YouTubeの動画URL（例: `https://www.youtube.com/watch?v=z2fPWh5pCQA`）を直接貼り付け**ておきます。
   - インポート時に、YouTubeの公式oEmbed APIを自動で呼び出し、**動画の正式なタイトルを取得して記事タイトルに設定**します。また、記事の上部に**レスポンシブなYouTube埋め込みプレイヤーが自動配置**されます。

#### ①-B. 既存の動画URL（4月〜6月）の自動マッピング手順 (url.md の一括自動配置)
Google Drive内のフォルダに `url.md` が存在しない場合、以下の手順で自動的にYouTube上の動画URLを照合・紐づけることができます。

1. **YouTubeの動画一覧データを取得する**
   - **方法A (APIキーあり)**: 
     環境変数 `YOUTUBE_API_KEY` を設定し、`python3 scripts/fetch_youtube_videos.py` を実行します。`scripts/youtube_videos_techmoney.json` および `scripts/youtube_videos_kabumaru.json` が生成されます。
   - **方法B (APIキーなし - 手軽)**: 
     ブラウザで YouTube チャンネル（TechMoney または かぶまる）の「動画」タブを開き、4月までの動画が表示されるように下までスクロールします。
     開発者ツール（F12）のコンソールを開き、以下のコードを実行して出力されたJSONをコピーします：
     ```javascript
     const videos = [];
     document.querySelectorAll('ytd-rich-grid-media, ytd-grid-video-renderer').forEach(el => {
       const titleEl = el.querySelector('#video-title-link, #video-title');
       if (titleEl) {
         const title = titleEl.innerText.trim();
         const url = titleEl.href;
         videos.push({ title, url });
       }
     });
     console.log(JSON.stringify(videos, null, 2));
     ```
     コピーしたJSONをそれぞれ `scripts/youtube_videos_techmoney.json` や `scripts/youtube_videos_kabumaru.json` として保存します。

2. **自動マッピングスクリプトの実行**
   環境変数 `GEMINI_API_KEY` を設定した状態で、以下のコマンドを実行します：
   ```bash
   python3 scripts/map_youtube_urls.py
   ```
   - 台本の内容やフォルダ名から、Gemini APIが最も合致するYouTube動画を自動照合し、フォルダ内に `url.md` を作成します。
   - 該当する動画が存在しない場合は、`none_url.md` が作成されます。

### ② タイトル・概要のカスタム調整 (`scripts/title_mappings.json`)
インポートする記事のタイトルや説明文、カテゴリーをコードに触れずに固定したい場合は、`scripts/title_mappings.json` に設定を記載します。
- 設定例:
  ```json
  {
    "hpe": {
      "title": "HPE 投資判断レポート - AIインフラ「オンプレミス回帰」の隠れた勝者",
      "description": "オンプレミスAI回帰で急浮上するHPEの強み、ジュニパー買収効果、バリュエーションを分析。",
      "category": "cloud"
    }
  }
  ```

### ③ インポートスクリプトの実行
準備が完了したら、以下のコマンドを実行します。
```bash
python3 scripts/import_youtube.py
```

- **`url.md` がある場合**: YouTube動画の正式なタイトルを取得し、記事の上部に動画プレイヤーを埋め込みます。
- **`url.md` がなく、`GEMINI_API_KEY` 環境変数がある場合**: Gemini API（`gemini-2.5-flash`）を使って、台本から魅力的なSEOタイトルと概要を自動でAI生成します。
- **それ以外の場合**: 概要ファイル（`gaiyou.md` 等）から、自動で正規表現を用いてタイトルと説明文を抽出します。

## 3. インポート完了後の検証

インポートが成功したら、以下のコマンドでコンパイルにエラーや route 衝突がないか確認してください。

```bash
npm run build:techmoney
```
エラーなく「Complete!」と表示されれば、本番公開の準備は完了です。
